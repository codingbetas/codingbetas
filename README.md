## 🧩 System Design & Architecture

### 🔹 Backend Architecture (Scalable System)

```text
        ┌──────────────┐
        │    Client    │
        │ (Web/Mobile) │
        └──────┬───────┘
               │
               ▼
        ┌──────────────┐
        │  API Gateway │
        │  (FastAPI)   │
        └──────┬───────┘
               │
     ┌─────────┼─────────┐
     ▼                   ▼
┌──────────┐       ┌────────────┐
│  Cache   │       │ Auth Layer │
│ (Redis)  │       │ JWT/OAuth2 │
└────┬─────┘       └────┬───────┘
     │                  │
     ▼                  ▼
┌──────────┐       ┌────────────┐
│ Database │       │ Background │
│PostgreSQL│       │ Workers    │
└──────────┘       │ (Celery)   │
                   └────┬───────┘
                        ▼
                 ┌────────────┐
                 │ Message    │
                 │ Queue      │
                 └────────────┘

```


### 🔄 Backend Flow
```
Client sends request to API
API checks Redis cache
Cache miss → fetch from PostgreSQL
Response cached for optimization
Background tasks handled asynchronously
```

     ▼                   ▼
┌────────────┐     ┌────────────┐

     ▼                  ▼

### 🔄 ML Flow
```

```
